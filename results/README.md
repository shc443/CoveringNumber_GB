# Results Directory

Use this folder for reproducible artifacts generated from scripts in this repo.

Suggested structure:

```
results/
  manifests/
  csv/
  figures/
  models/
```

Recommended metadata per run:

- command
- config path
- git commit hash
- random seed
- start/end timestamp
- hardware (GPU/CPU and memory)

For the refactored framework, outputs are created under:

`cleaning/curse_dimensionality_sim/results/`
