How DNNs Break the Curse of Dimensionality

Compositionality and symmetry learning with reproducible synthetic experiments.

Paper Code Colab Dataset

Abstract

This project studies how deep architectures exploit compositional structure f = h ∘ g to avoid shallow-model sample complexity scaling in high dimensions. The repository includes original notebooks and a refactored framework for reproducible sweeps over smoothness parameters, sample sizes, and architecture families.

Compositional Data

Matérn kernel sampling controls smoothness for both g and h components.

Architectures

Accordion, Deep, and Shallow baselines are compared under a unified training schedule.

Theory Link

Empirical results are paired with complexity metrics and bound-performance diagnostics.

Method Pipeline

Sample Matérn g
→
Compose Y = h(g(X))
→
Train Acc/Deep/Shallow
→
Analyze loss + complexity

pip install -r cleaning/curse_dimensionality_sim/requirements.txt
cd cleaning/curse_dimensionality_sim
python experiments/run_compositionality_study.py --focus
