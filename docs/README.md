# GitHub Pages Setup

This project page is served from `docs/` and deployed by:

`/.github/workflows/pages.yml`

## One-Time Repository Setting

1. Go to **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Save.

Expected URL:

`https://shc443.github.io/CoveringNumber_GB/`

## Updating Page Content

- Edit `docs/index.html` for text and structure.
- Edit `docs/styles.css` for visual style.
- Add figures under `docs/assets/` and reference them from `index.html`.
- Push to `main` to trigger deployment.
