# Project Page Assets

Place project-page visual assets in this directory.

Suggested files:

- `hero_overview.png`
- `heatmap_accnet.png`
- `learning_curve.png`
- `complexity_correlation.png`

When you add files, reference them from `docs/index.html` using relative paths,
for example:

```html
<img src="assets/hero_overview.png" alt="Method overview">
```
